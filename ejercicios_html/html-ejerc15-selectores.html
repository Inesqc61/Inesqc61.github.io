Tipos de selectores que entran en examen

Selector de Clase (class="")

En un documento HTML, los selectores de clase buscan un elemento basado en el contenido de su atributo class. El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.

Selector de Identificador (id="")

Este párrafo tiene un identificador único que cambia su color a naranja y lo pone en cursiva. El selector de identificador se usa para aplicar estilos a un elemento específico en la página.

Selector de Todos los elementos (*)

Este selector aplica un estilo a todos los elementos de la página. En este caso, se ha cambiado la fuente a Arial para todo el documento.

Selector de Varios elementos (p, em, h1, h2)

Este selector aplica estilos a múltiples elementos a la vez. Aquí se ha aplicado un margen de 10px arriba y abajo a los elementos <p>, <em>, <h1> y <h2>.

Selector de Elementos contenidos en otros elementos (ol li)

  1. Este es un elemento de lista dentro de una lista ordenada.
  2. El selector ol li aplica un color azul a todos los elementos <li> dentro de un <ol>.

Selector de Sobrevuelo (p:hover)

La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor (puntero del mouse).
